# Documentation
Documentation for ZPI Project

# How to open documentation:
 1. use pip install -r requirements.txt
 2. use `sphinx-quickstart`
 3. `make html`   
 4. `open build/html/index.html`

# How to add chapter:
 1. add .rst file to source folder
 2. add .rst file to index.rst
 3. `make html`   
 4. `open build/html/index.html`
